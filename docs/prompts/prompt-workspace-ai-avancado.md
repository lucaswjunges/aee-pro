# Prompt Original — Workspace AI Avançado (2026-02-23)

Vamos começar com o desenvolvimento de uma página no site que permita a livre geração e edição de documentos LaTeX assistido por inteligência artificial. Quero algo parecido com o Claude Code, com possibilidade inclusive de haver um diretório, onde se pode anexar arquivos nos prompts e etc. Quero algo avançado mesmo. Aí a IA compila, recebe o log de compilação, faz edições, manda compilar de novo se tiver dado erro ou warning, etc. Quero algo bem pensado, e a professora pode pagar Claude Code ou outra coisa. Se já tiver algum software assim pronto, melhor ainda, tipo um CLI do Gemini, codex, claude code, etc. Aí a professora ainda pode gerar vários documentos e escolher um principal, tendo inclusive um diretório na nuvem. Defina bem qual nuvem vamos usar pensando na análise custo-risco-benefício. O que é inteligente de se usar sabendo que a professora agora também quer poder colocar fotos e vídeos de seus aluno no AEE+ Pro? Essa "ferramenta" também vai ser usada para a geração de jogos que podem ser impressos, material de aula, slides, enfim, um imenso uso livre e liberdade criativa. Quero algo que seja robusto e profissional. Vamos fazer com as melhores opções que temos e que possamos pagar. Antes, relembre como o sistema funciona e tecnologias e serviços que usa. Ou se é melhor simplesmente usar o banco de dados que já temos e processamento no javascript. Não sei. Inclusive com geração de imagens e vídeos. Mas quero algo inteligente de verdade, que consiga ter contexto, lidar com documentos grandes e ser precisa. E que tenha uma abordagem do tipo que quer auxiliar o usuário a ele obter o que quer, mas ao mesmo tempo ser simples e funcional, como se tivesse sido Steve Jobs que projetou. Dica: você pode remodelar o sistema todo se achar que ficaria melhor. Inclusive pode tratar os tipos de arquivos atuais como arquivos e o seu história de edições ser os PDFs anteriores do mesmo tipo e tratar como um ícone num diretório virtual, se preferir. Tem que ver se vale a pena usar o próprio Google Drive (quase todo usuário tem), se vale a pena "ser um parceiro Google" e usar nuvem deles, integração com google documentos, ou se vai ter problemas. Também tem que fazer uma revisão do estado-da-arte disso em 2026. Quero inclusive a capacidade de instalar softwares linux e usá-los, tipo como o Claude Code faz. Se você quiser e achar que fica melhor, as capacidades mais avançadas poderiam estar disponíveis caso usuário entre no modo avançado, pois muitas professoras têm dificuldade com tecnologia. Na minha opinião ficaria muito bom se tivesse Claude Code --dangerously-skip-permissions no navegador de alguma forma, e uma interatividade mais amigável e direcionada com ele para as professoras. Isso também resolveria a dificuldade que temos com autofix no sistema AEE+ Pro e o Claude Code simplesmente não tem. Algo tão bom que até eu usaria quando estou sem meu notebook, e que poderia futuramente virar um produto a parte. Lembrando que vai ser a professora que vai arcar com os custos desse processamento e billing. Então vou poderia desfrutar disso também, e não precisar mais pagar Claude Code. E não quero algo muito demorado de usar, quero uma boa experiência. E registre esse prompt também, num arquivo .md. Também quero uma boa gestão e planejamento disso tudo.

## Requisitos Extraídos

1. **Workspace de documentos LaTeX** com IA assistente (estilo Claude Code no browser)
2. **Diretório virtual** com upload de arquivos, anexos em prompts
3. **Compilação LaTeX automática** com loop de fix (compila → lê log → corrige → recompila)
4. **Armazenamento em nuvem** para fotos, vídeos, documentos
5. **Geração de imagens e vídeos** via IA
6. **Jogos imprimíveis, material de aula, slides** — uso livre e criativo
7. **Multi-provider AI** — usuário paga própria chave (Claude, GPT, Gemini)
8. **Modo simples** (professoras) + **Modo avançado** (power users)
9. **Capacidade de executar comandos/softwares** (sandbox tipo Claude Code)
10. **Integração com Google Drive** (avaliar custo-risco-benefício)
11. **Histórico de versões** de cada documento
12. **LGPD compliance** para dados de menores
13. **Potencial produto standalone** no futuro
14. **Billing direto ao usuário** (professora paga seu próprio uso de IA)
